## Study Badge란?
여러 스터디 모임을 조회하고 생성, 관리할 수 있는 플랫폼입니다.

꾸준한 스터디 참여가 힘들다면 스터디뱃지로 성공해보세요!

출석률에 따른 💰벌금/보상 시스템💰이 적극적인 참여를 도와줍니다.

일정 등록, 출석 체크, 출석률에 따른 벌금과 보상 기능, 

이달의 출석왕 선정 등으로 스터디 모임을 적극적이고 재밌게 진행해보세요!
